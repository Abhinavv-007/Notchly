# Contributing to Notchly

Notchly is maintained by Abhinav Raj. Pull requests are welcome.

- Open an issue first for non-trivial changes so we can talk through the approach.
- Run the build locally before submitting: `xcodebuild -project Notchly.xcodeproj -scheme Notchly -configuration Debug build CODE_SIGNING_ALLOWED=NO`.
- Match the existing Swift / SwiftUI style.

Questions: [abhnv.in](https://abhnv.in).
